我从小就非常喜欢儿童绘本，尤其是机关立体绘本。
小朋友翻动书页、用手指推拉纸片，画面就发生神奇的变化，充满快乐和想象力！小小的我一个人可以玩一本书玩半天。
我也因此喜爱上了绘画、手工和艺术。
在小红书上我看到很多妈妈分享他们的宝宝，和童年的我一样对可动绘本爱不释手，玩一本书可以咯咯笑个不同。
不过好的绘本都价格不菲，一个绘本通常只有一种核心玩法。

于是我想，为什么不用vibe-coding做一个可以让宝宝用身体操控和感知的“体感绘本”呢？
每一次举手，画面可以发生戏剧性的变化；
每一次推动，颜色可以发生奇妙的融合。
宝宝对着屏幕（最好是更大的电视屏），探索色彩、形状，探索各种未知。抽象的"色彩"“形状”转化为了一种具象的"身体记忆"，皮亚杰认知发展理论中，"感知运动阶段"的学习特征的宝宝正需要这样的“玩耍”。

每一次挥手都是一次主动探索，每一次色彩融合都是一次惊喜发现——玩，就是最好的学习。

产品基于浏览器端实时人体姿态识别（MediaPipe），无需安装App、无需外接手柄，任何带摄像头的设备打开网页就可以体验。全程在本地运行，保护宝宝的隐私安全。
